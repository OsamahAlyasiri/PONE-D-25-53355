# PONE-D-25-53355
PLOS-ONE-DATASETES-SHARING
Here, we provide access to benchmark datasets for multi-class imbalanced text classification used in our PhD research.

This repository includes term-weighted versions of several well-known text collections, where the original datasets have been processed using the proposed enhanced supervised term weighting schemes, namely Modified Term Frequency–Modified Inverse Class Frequency (MTF-MICF).

It is important to note that the original datasets were not created by the author. Instead, this work provides transformed versions of these datasets after applying the proposed weighting schemes, with the following objectives:

1. To enable fair and reproducible comparisons among different text classification approaches. Although these benchmark datasets are widely used, prior studies often report variations in feature representations and term counts. By providing standardized term-weighted versions, this resource ensures consistency and improves the reliability of comparative evaluations.

2. To facilitate research and experimentation, particularly for new researchers in the field. The provided datasets are preprocessed and structured in a simplified format, reducing the effort required for data preparation and allowing researchers to focus on modeling and evaluation. 

3. To support research on multi-class imbalanced text classification. Some of the original datasets have been adapted to explicitly reflect imbalanced class distributions, making them suitable for evaluating methods designed to handle imbalance.

These datasets are shared under the same terms as their original sources and are intended for research purposes only. For any other use, permission must be obtained from the original dataset creators. Links to the original sources are provided where applicable.

If you use these datasets in your work, please cite the corresponding publications.
